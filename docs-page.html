What is iPlantbox? Last updated: 2023-10-20

iPlantbox is a cutting-edge innovation that transcends traditional gardening, heralding a new era of intelligent plant cultivation. Designed to harness the power of advanced hardware technology and intelligent control systems, iPlantbox orchestrates and maintains the ideal growth environment for plants within a confined space, ensuring that every aspect of their development is optimized.

Key Features

  • Advanced Hardware Integration
  • Real-time Sensing
  • Dynamic Adjustments
  • Ideal Growth Atmosphere
  • Sustainability and Resource Efficiency

Plantbox Setups

It is where you can configure the settings for your plantbox.

Soil Settings

Soil Setup Page, wherein you can configure the soil settings for your project. Proper soil management is crucial for the success of your crops and plants.

Soil Setup Form

Here, you can set various parameters to create an ideal soil environment.

  • Soil Name – An identifier or unique name for a particular soil sample, helping you distinguish between different soil types or samples.
  • Nitrogen Detail – Users can specify the ideal nitrogen content for this soil sample. Nitrogen is a critical nutrient for plant growth.
  • Phosphorus Detail – The ideal phosphorus content of the soil sample.
  • Potassium Detail – The ideal potassium content of the soil sample.
  • Soil Description – A brief description of the soil sample, which might include information like its origin, characteristics, or any special properties.

Soil Setup List

This is a table displaying the records of ideal soil environments that have been configured using the Soil Setup Form. Each row in the table represents a different soil sample and includes the details you've collected in the form.

Crop Type Settings

Here, you can set various parameters to configure the settings for your crop type.

Crop Type Setup Form

Here, you can define various parameters to create and manage different crop types.

  • Crop Type Name – This field is for providing a unique identifier or name for the crop type. It helps users distinguish between different types of crops.
  • Crop Description – This field is for a brief and consistent description of the crop type. Users can provide additional information about the characteristics or features of the crop.

Crop Type Setup List

This is where users can view a list of the configured crop types. It displays the names and descriptions of the different crop types that have been added using the Crop Type Setup Form.

Crop Setting

The Crop Setup allows users to manage and keep track of different crops, ensuring they have the necessary information to customize settings and monitor the progress of each crop.

Crop Setup Form

Here, you can define the parameters for managing individual crops.

  • Crop Name – This is where you enter a unique name or identifier for the crop. This name helps distinguish one crop from another.
  • Crop Type – You can select the appropriate crop type from a dynamic combo box. The combo box provides a list of available options, and this list can change based on the provided Crop Type Setup.

Crop Setup List

The Crop Setup List is a table that displays records of crop details. Each row in the table represents a different crop and includes information such as the crop name and crop type.

Ideal Environment Settings

Ideal Environment Setup page, where you can configure the optimal conditions for your crops to thrive. Creating the right environment is essential for maximizing growth, health, and productivity.

Ideal Environment Setup Form

Here, you can customize various parameters to create the perfect conditions for your specific plants.

  • Ideal Environment Name - Provide a name or identifier for the ideal environment setup. This name helps users distinguish between different configurations.
  • Select Soil - You can select the appropriate soil type from a dynamic combo box. The combo box provides a list of available options, and this list can change based on the provided Soil Setup.
  • Select a Crop - You can select the appropriate crop from a dynamic combo box. The combo box provides a list of available options, and this list can change based on the provided Crop Setup.
  • Enter Ideal Temperature – Set the ideal temperature for the plant box to ensure the best growth conditions.
  • Enter Ideal Humidity - Specify the desired humidity level, which is crucial for plant health.
  • Enter Ideal Soil Moisture - Define the optimal soil moisture content for the plants.
  • Enter Ideal Tank Water Level - Set the ideal water level in the tank, as water is essential for plant growth.

Ideal Environment Setup List

The Ideal Environment Setup List is a feature that displays a list of the ideal plant environments that users have configured. It serves as a summary and quick reference to the various setups created for different plants or crops.

Project Setups

It is where you can configure the settings for your plantbox.

Plantbox Setup Form

Here, you can define parameters for managing individual plant boxes.

  • Plantbox Name - Here, users can provide a unique name or identifier for the plant box. This helps distinguish one plant box from another.
  • Plantbox Description - Users can add a brief description of the plant box, which may include details about its location, contents, or purpose.

Plantbox Setup List

The Plantbox Setup List is a table or list that displays records of plant box details. Each entry in the list represents a different plant box and includes the name and description of the plant box.

Start a Project

The implementation of this setup will initiate the commencement of the selected Environment Setup.

Start a Project Form

Here, you can start a project and associate it with an ideal environment setup.

  • Enter Project Name – Here, users can provide a name or identifier for the project, making it easier to distinguish between different projects.
  • Select a Plant Box – This is a dropdown menu (combo box) where users can choose a plant box from a list created in the "Plantbox Setup" section. It allows users to associate a specific plant box with the project.
  • Select Configured Setup for the Project – This is another dropdown menu (combo box) that enables users to select an ideal environment setup for the project. Users can choose from the list of setups created in the "Ideal Environment Setup" section.

Project List

The Project List is a feature that displays a list of projects initiated by users. This list typically provides a quick overview of each project.

Plant Box Setup Instructions

A. Filling the Soil Container

  1. Remove the soil container from inside the plant box.
  2. Fill the soil container to the top.
  3. Plant or transplant your desired crop in the center of the soil container.
  4. Reinsert the soil container inside the plant box.
  5. Insert 80% of the metal probe from the soil moisture sensor into the ground.
  6. Insert the arrowhead of the water irrigation system into the soil.

We recommend checking the soil moisture sensor's probe weekly for any signs of corrosion.

B. Adding Water

  1. Open the water tank enclosures located on the sides of the plant box.
  2. The left water tank is for water irrigation, and the right water tank is for the humidifier.
  3. Open the water tank lid.
  4. Add water to the water tank.
  5. Close the lid.
  6. Close the water tank enclosures.

Please fill the water tanks to 80% of their capacity. Refer to the system's dashboard to check the actual water level.

C. Connecting Auxiliary and Peripheral Devices

  1. Ensure that the power cord is unplugged.
  2. Check the labels and find the corresponding ports.
  3. Connect the auxiliary and peripheral devices to the matching ports.

D. Switching On/Off Mode

  1. Plug the control box into a 220-volt outlet.
  2. Turn on the main switch.
  3. Activate the sensor switch.
  4. Activate the gatherer switch.
  5. Activate the system switch.

Follow this sequence to ensure a proper system initialization.

E. System Initialization

  1. After switching on the plant box, wait for an SMS confirmation.
  2. The "Connected to the Wi-Fi network" and "Global variables Retrieved and Set" notification indicates a successful plant box initialization.
  3. A "Wi-Fi Connection Failed" notification means the plant box could not connect to the internet and initialize.

The plant box will attempt to connect to the internet three times. If unsuccessful, it will restart and repeat the system initialization process.

F. Device Care and Maintenance

  1. Perform a weekly assessment of the device for any signs of damage.
  2. Weekly, review sensor readings and recalibrate if adjustments are necessary.
  3. On a weekly basis, monitor the SIM card's load balance and reload it as needed.
